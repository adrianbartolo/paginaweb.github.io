WEB

Objetivo

Haciendo uso de un editor de texto (HTML), el ingeniero aprendio y desarrollo las tecnicas suficientes para la elaboracion de paginas webs estaticas y al mismo tiempo a desarrollar paginas webs dinamicas, incluyendo el uso del lenguaje de programacion PHP y MySQLi para la conexion de Bases de datos y la pagina web en dado caso de que sea necesario, involucrando un poco el desarrollo de funciones en Java Script.
